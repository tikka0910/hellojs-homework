>####week1使用筆記 (git版控)
***

```
 1. fork
 2. $git clone url //clone回本機
 3. $git init //初始化專案
 4. $git config --global user.name "XXXX" 
 5. $git config --global user.email XXXX@example.com
 6. $git add .  //加入所有檔案到索引(index)
 7. $git commit -m "訊息" //提交檔案到本端
 8. $git remote add XXX url //建立一個遠端名為XXX
 9. $git push   //push到遠端
 10. pull request

```
