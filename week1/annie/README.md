#Week1 心得


>雖然過去在學校，有使用GitLab進行程式作業的繳交，但一直不太了解它的用意，
經由這次上課學習到關於協同合作和版本控制的重要性及各式軟體，
感覺十分新奇，讓人有學習新技術的興奮感。不過上課速度有些快速，加上偶爾螢幕較低位置的部分會被同學的頭擋到而漏掉步驟，
過程中有些緊張，
但實習生和講師都非常有耐心解決我們的疑問，
希望在這將近快二個月的課程當中，能更充實自己的能力。
